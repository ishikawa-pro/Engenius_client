# Engenius
Engeniusは話題の技術ブログの記事を、一つのアプリで閲覧したいと思い作ってみました。
名前は技術者(Engineer)の賢い(Genius)情報収集アプリを目指しているので、
Engineer + Genius = Engeniusです笑
